## What's changed

- **Chrome Web Store compliance** — shortened the manifest `description` to **114 characters**
  (the store rejects anything over **132**; v1.0.0's was 161). Purely metadata — no behavior,
  files, or permissions changed.

Everything else is as shipped in v1.0.0 — see [`v1.0.0.md`](v1.0.0.md) for the full feature
list, file map, permission justifications, and the submission checklist.
